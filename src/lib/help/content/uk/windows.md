# Вікна
{app} використовує окремі вікна для різних завдань.

## 🏷  Вікно мітки
Відкривається через меню, клавіші або кнопку. Введіть мітку для позначення поточного моменту ЕЕГ.

## 🔍  Вікно пошуку
Вікно пошуку має три режими — схожість ЕЕГ, текст та інтерактивний — кожен з яких запитує ваші дані по-різному.

## Пошук за схожістю ЕЕГ
Оберіть діапазон дат/часу і запустіть пошук найближчих сусідів по всіх ZUNA-embeddings, записаних у цьому вікні. Індекс HNSW повертає k найбільш схожих 5-секундних епох ЕЕГ з усієї вашої історії, відсортованих за косинусною відстанню. Менша відстань = більш схожий стан мозку. Мітки, що перекривають часову позначку результату, відображаються одразу.

## Текстовий embedding-пошук
Введіть будь-яке поняття, діяльність або ментальний стан природною мовою (напр. «глибока концентрація», «тривожність», «медитація із заплющеними очима»). Запит перетворюється на вектор тією ж моделлю sentence-transformer, що використовується для індексації міток, і зіставляється з усіма вашими анотаціями за косинусною схожістю в індексі HNSW. Результати — ваші власні мітки, відсортовані за семантичною близькістю, а не за збігом ключових слів. 3D kNN-граф візуалізує структуру сусідства.

## Інтерактивний кросс-модальний пошук
Введіть поняття у вільному тексті, і {app} запустить чотириетапний кросс-модальний конвеєр: (1) запит перетворюється на вектор; (2) знаходяться text-k семантично найближчих міток; (3) для кожної мітки {app} обчислює середній ЕЕГ-вектор за її вікном запису та шукає eeg-k найбільш схожих епох ЕЕГ; (4) для кожного сусіда ЕЕГ збираються анотації в межах ±reach хвилин як «знайдені мітки». Результат — спрямований граф з чотирьох шарів — Запит → Текстові збіги → Сусіди ЕЕГ → Знайдені мітки — як інтерактивна 3D-візуалізація, яку можна експортувати у SVG або Graphviz DOT.

## 🎯  Вікно калібрування
Виконує кероване калібрування. Потребує підключеного та активного пристрою BCI.

## ⚙  Вікно налаштувань
Чотири вкладки: Налаштування, Комбінації (глобальні гарячі клавіші, палітра команд, клавіші в додатку), Модель ЕЕГ (кодувальник та статус HNSW). Відкрити з меню трею або кнопки шестерні.

## ?  Вікно довідки
Це вікно. Повний довідник інтерфейсу {app}.

## 🧭  Майстер налаштування
П'ятикроковий майстер першого запуску: підключення Bluetooth, перевірка посадки та перше калібрування. Відкривається автоматично при першому запуску; можна відкрити знову з палітри команд (⌘K → Майстер налаштування).

## 🌐  Вікно статусу API
Панель у реальному часі з усіма підключеними клієнтами WebSocket та журналом запитів. Показує порт сервера, протокол та виявлення mDNS. Містить фрагменти швидкого підключення для ws:// та dns-sd. Автооновлення кожні 2 секунди. Відкрити з меню трею або палітри команд.

## 🌙 Стадії сну
Для сесій від 30 хвилин відображається автоматично згенерована гіпнограма. Примітка: побутові BCI-гарнітури на кшталт Muse мають 4 сухі електроди — стадіювання є наближеним, це не клінічна полісомнографія.

## ⚖  Порівняти
Виберіть два часові діапазони на часовій шкалі та порівняйте розподіли потужності діапазонів, показники розслаблення/залученості та FAA поруч. Включає фази сну, розширені метрики та Brain Nebula™ — 3D UMAP-проєкцію, що показує схожість двох періодів у багатовимірному EEG-просторі. Відкрити з меню трею або палітри команд (⌘K → Порівняти).

# Оверлеї та палітра команд
Оверлеї швидкого доступу, доступні в кожному вікні через комбінації клавіш.

## ⌨  Палітра команд (⌘K / Ctrl+K)
Випадний список швидкого доступу з усіма доступними діями додатку. Введіть для фільтрації, ↑↓ для навігації, Enter для запуску. Доступний у кожному вікні. Команди включають відкриття вікон (Налаштування, Довідка, Пошук, Мітка, Історія, Калібрування), дії з пристроєм (повторити з'єднання, налаштування Bluetooth) та утиліти (показати комбінації, перевірити оновлення).

## ?  Оверлей комбінацій клавіш
Натисніть ? у будь-якому вікні (поза текстовими полями) для перемикання плаваючого оверлею з усіма комбінаціями клавіш — глобальні комбінації з Налаштування → Комбінації, плюс клавіші в додатку як ⌘K для палітри команд та ⌘Enter для збереження міток. Натисніть ? знову або Esc для закриття.
