### Features

- **Neurable MW75 Neuro headphone support**: Added full session support for the Master & Dynamic MW75 Neuro EEG headphones (12 channels at 500 Hz). The `mw75` crate (with `rfcomm` feature) is added to `skill-devices` and re-exported. The MW75 connection follows the same lifecycle as the `mw75` CLI binary: BLE discover → BLE connect → BLE activate (EEG + raw mode) → disconnect BLE → RFCOMM stream on channel 25. The headphones must first be paired via OS Bluetooth settings (hold power button 4+ seconds). BLE scanner discovers MW75 by name and GATT service UUID (with periodic targeted scans for macOS paired devices). When no preferred device is set, `client.connect()` finds and connects in one step; otherwise `scan_all()` + `connect_to()` targets the exact peripheral. DSP pipeline processes first 4 of 12 channels; all written to CSV. Battery, DND, band enrichment fully integrated. MW75 constants added to `skill-constants`.
